# Sistema de Tarefas em Console

Projeto simples desenvolvido para praticar lógica de programação e estruturas básicas.

## Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa

## Conceitos utilizados
- Variáveis
- Estruturas de dados
- Condicionais
- Laços de repetição
- Funções

## Objetivo
Consolidar fundamentos de programação por meio de um projeto prático.
